# Navigant

Navigant Technologies - Business Process Outsourcing & IT Services Website

## About

Navigant Technologies is a global outsourcing firm providing comprehensive BPO services, IT solutions, and digital transformation services.

## Project Structure

```
Navigant/
├── client/          # React + TypeScript frontend application
│   ├── src/         # Source code
│   ├── public/      # Static assets
│   └── package.json # Dependencies
└── README.md        # Project documentation
```

## Technologies

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
cd client
npm install
```

### Development

```bash
cd client
npm run dev
```

### Build

```bash
cd client
npm run build
```
